## Snake

This is a fun interview challenge for Typescript.

## Run

To start a new game run:

`yarn start {height} {width}` 

## Test

`yarn test`

## Conventions

The grid is specified in (x, y) coordinates i.e.

```
 ----------> x
|
|
|
|
y 
```
